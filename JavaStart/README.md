# KB_study_JAVA

KB IT's Your Life 7기 부트캠프 과정에서 학습한 **Java 실습 및 복습 코드**를 정리하는 저장소입니다.

이 저장소는 수업 시간에 진행한 예제, 교안 기반 실습, 문법 복습, 그리고 객체지향 개념을 다시 확인하기 위한 코드를 기록하는 용도로 사용합니다.

---

## 저장소 목적

- 부트캠프 Java 수업 내용을 꾸준히 기록하기
- 수업에서 배운 문법과 개념을 직접 다시 구현해보기
- 실습 과정에서 작성한 코드를 정리하고 복습하기
- 이후 알고리즘, DB, 백엔드 학습으로 이어질 Java 기초를 다지기

---

## 학습 방향

현재는 Java 백엔드 학습을 위한 기초 체력을 만드는 것을 우선 목표로 두고 있습니다.

주요 학습 범위는 다음과 같습니다.

- 클래스 / 객체 / 생성자
- 상속 / 다형성 / 오버라이딩
- 추상 클래스 / 인터페이스
- 예외 처리
- 컬렉션(List, Set, Map)
- 제너릭 기초
- 파일 입출력 기초

---

## 저장소 구성 방향

이 저장소에는 아래와 같은 성격의 코드가 기록됩니다.

- 수업 시간 예제 코드
- 교안 또는 책 기반 실습 코드
- Java 문법 복습 코드
- 객체지향 개념 확인용 예제
- 실습 제출 전 테스트 코드

예정 학습 주제에 따라 폴더 구조는 계속 정리 및 확장될 수 있습니다.

예시:

```text
KB_study_JAVA/
├─ ch01/
├─ ch02/
├─ ch03/
├─ oop/
├─ exception/
├─ collection/
└─ README.md
```

# 기록 원칙

- 배운 내용은 가능한 직접 다시 작성해본다.
- 모르는 개념은 정리만 하지 않고 코드로 확인한다.
- 작은 코드라도 꾸준히 commit 하며 기록을 남긴다.
- 단순 복붙보다 “왜 이렇게 동작하는지” 이해하는 것을 우선한다.

직접 구현하고 설명할 수 있는 수준까지 끌어올리는 것을 목표로 학습하고 있습니다.
